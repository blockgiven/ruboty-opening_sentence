# Ruboty::OpeningSentence

Ruboty plugin for 小説の書き出し.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ruboty-opening_sentence'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ruboty-opening_sentence

## Usage

    @ruboty 書き出し
